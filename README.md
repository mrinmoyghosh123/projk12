# projk12.org

Static "coming soon" landing page for [www.projk12.org](https://www.projk12.org).

Served via GitHub Pages from `main` / root.

## Files

- `index.html` — the live page
- `CNAME` — custom-domain mapping for GitHub Pages (`www.projk12.org`)
- `Landing Page v1.html` — earlier draft, kept for reference

## Updating

Edit `index.html`, then:

```bash
git commit -am "Update landing copy"
git push
```

GitHub Pages re-deploys in ~30 seconds.
